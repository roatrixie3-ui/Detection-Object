# 🎥 Live Object Detection & Tracing

A real-time object detection web application built using:

- Python
- Streamlit
- YOLOv8
- OpenCV
- streamlit-webrtc

This app uses your webcam to detect and track objects live in the browser.

---

# 📌 Features

✅ Real-time object detection  
✅ Object tracking using YOLOv8  
✅ Webcam support  
✅ Fast and lightweight  
✅ Browser-based interface  

---

# 🛠 Requirements

Make sure you have:

- Python 3.9+
- pip installed
